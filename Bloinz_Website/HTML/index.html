Projekt Informationen

Spielbeschreibung

In Bloinz 2D muss man als kleiner Blob bösen Münzen und Kanonenkugeln ausweichen, während man gute Münzen sammelt, um Punkte zu erlangen. Dabei unterstützt einen der Randomcoin, welcher eine zufällige positive Aktion ausführt, wie beispielsweise einen Münzmagneten, welcher gute Münzen anzieht.

Der Spieler startet bei einer mittleren Größe. Um zu gewinnen, müssen genügend positive Coins eingesammelt werden, um die Größe 25 zu erreichen. Dann ist das Spiel gewonnen. Aber aufgepasst, wenn die negativen Münzen eingesammelt werden, gibt es Minuspunkte und der Blob wird kleiner. Sobald der Blob die Größe 0 erreicht, ist das Spiel leider verloren.

Zusätzlich bietet Bloinz 2D die Möglichkeit, die Spielfiguren sowohl im Aussehen als auch im Verhalten zu verändern. Es gibt die Möglichkeit, das Design ganz nach seinen eigenen Wünschen anzupassen. Auch das Verhalten, wie beispielsweise die Sprungkraft oder Geschwindigkeit, lassen sich umstellen. Dies geht sowohl in den Einstellungen, als auch im eigentlichen Script des Spiels, wodurch grobe Einblicke in die Welt der Programmierung gegeben werden.

Beschreibung des Projekts

Bei dem Projekt Bloinz2D handelt es sich um eine Spielanwendung, an welcher grundlegende Prinzipien der Programmierung erklärt werden können. Das Projekt wurde mit der Game-Enging Godot umgesetzt.
Die grundlegenden Programmierprinzipien wie Variablen und Methoden können direkt am Code nachvollzogen werden und gleichzeitig auch bearbeitet werden. So kann man die Werte einer Variablen anpassen, um sich das Spiel einfacher zu machen, oder eine ganze Methode umschreiben, um gegebenenfalls andere Features noch zu integrieren.
Auch Objektorientierung ist aus dem Spiel herzuleiten. Die Vererbung von Eigenschaften wird bei den Münzen deutlich. So gibt es eine allgemeine Münz-Elternklasse, von welcher dann alle einzelnen Münzen jeweils abgeleitet werden und mit ihren speziellen Eigenschaften ergänzt werden. Auch dies lässt sich im Code Editor nachvollziehen.
Eine weitere essenzielle Eigenschaft des Projekts ist das Anpassen des Aussehens der Spielfiguren. Somit lässt sich das Spiel in die unterschiedlichsten Stile verändern, ganz nach der eigenen Kreativität.

Game-Engine Godot

Das Spiel wurde in der Game-Engine Godot erstellt. Diese ermöglicht eine schnelle und einfache Programmierung für Spielanwendungen. Godot bietet eine einfache Übersicht über unterschiedliche Szenen, um in diesen die gewünschten Spielobjekte zu erzeugen. Zusätzlich bietet GODOT eine eigene eingebundene Programmiersprache: GD-Script, über welche auch das Spiel Bloinz 2D komplett programmiert wurde.

Nachdem du unser kleines Spiel zu grundlegenden Programmier-Prinzipien hoffentlich ausführlich auseinander genommen hast, kannst du auch selbst eine kleine Spielanwendung erstellen.
