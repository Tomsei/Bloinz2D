Bloinz 2D

Projekt Informationen

Beschreibung des Projekts

Bei dem Projekt Bloinz2D handelt es sich um eine Spielanwendung an welche Grundlegende Prinzipien der Programmierung erklärt werden können. Das Projekt wurde mit der Game-Enging Godot umgesetzt
Grundlagen der Objektorientierung werden durch das Spiel dargestellt. Sowohl Variablen, Methoden als auch grundlegende Vererbung ist in dem Spiel erkennbar.

Spielbeschreibung

Es handelt sich um ein einfaches Spiel in dem es darum geht, dass eine Spielfigur unterschiedliche Münzen einsammeln kann, welche jeweils unterschiedliche Auswirkungen auf den Spieler haben. Es gibt poitive Münzen, welche die Spielergröße erhöhen und negative welche sie verringert. Sobald der Spieler zu klein geworden ist, ist das Spiel verloren. Das Gewinnen eines Spiels ist vergleichbar. Hierzu muss der Spieler eine bestimmte Spielergröße erreicht haben.
Zusätzlich zu den Normalen Münzen gibt es noch die Zufallsmünzen. Diese lösen zufällig ausgewählte Aktionen aus, welche das Spiel Beeinflussen. Als positive Aktion zählt beispielsweise der Münzagnet, welcher die positiven Münzen zum Spieler zieht. Eine Negative Aktion ist beispielsweise ein Coin Regen von negativen Münzen

Game-Engine Godot

Das Spiel wurde in der Game-Engine Godot erstellt. Diese ermöglicht eine schnelle und einfache Programmierung für Spielanwendungen.
Godot bietet eine einfache Übersicht über unterschiedliche Szenen um in diesen die gewünschten Spielobjekte zu erzeugen
Zusätzlich bietet GODOT eine eigene eingebundene Programmiersprache GD-Script über welche auch das Spiel Bloinz 2D komplett programmiert wurde

Nachdem du unser kleines Spiel zu grundlegenden Programmier-Prinzipien hoffentlich ausfürhlich auseinander genommen hast, kannst du auch selbst eine kleine Spielanwendung erstellen
